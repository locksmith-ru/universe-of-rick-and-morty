# universe-of-rick-and-morty
Клиентское приложение к онлайн сервису https://rickandmortyapi.com/

- Toolkit Jetpack Compose
- Hilt
- Compose navigation
- Caching data with Room database
- adaptive design (phone, tablet, desktop)
